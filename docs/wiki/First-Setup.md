# First Setup

## 1. Get an OpenRouter API Key

Darce uses [OpenRouter](https://openrouter.ai) to access 200+ AI models with a single API key.

1. Go to [openrouter.ai/keys](https://openrouter.ai/keys)
2. Sign up or log in
3. Create a new API key
4. Copy it (starts with `sk-or-v1-...`)

## 2. Paste Your Key

On first launch, Darce shows the onboarding screen:

1. Paste your OpenRouter API key
2. Click **Connect**
3. Done — you're ready to code

## 3. Open a Project

- **File → Open Folder** (or `Ctrl+O`)
- Select any project folder
- Darce loads the file tree and you can start chatting

## 4. Optional: Web Browsing

To let the agent browse web pages and search Google:

1. Go to **File → Preferences**
2. In the **Web Browsing** section, paste a [CrawlRocket](https://crawlrocket.com) API key
3. Save

## 5. Optional: BrowserOS

For full browser automation (clicking, filling forms, testing UI):

1. Download [BrowserOS](https://github.com/browseros-ai/BrowserOS/releases)
2. Open BrowserOS → Settings → Enable MCP server
3. In Darce: Preferences → Toggle **Browser** ON
